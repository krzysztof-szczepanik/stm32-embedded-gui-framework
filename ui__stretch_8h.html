Detailed Description

Stretching behavior for widgets inside layout containers.

This module defines stretch policies used by layout containers such as HBox, VBox, Header, Footer, and others.

Stretching determines how a widget expands to fill available space along the horizontal and/or vertical axis.

These values are used by container layout engines during size allocation.
