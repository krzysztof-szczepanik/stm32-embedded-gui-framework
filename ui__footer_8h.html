Detailed Description

Footer section widget used at the bottom of application screens.

The footer is a semantic UI component placed at the bottom of a screen. It typically contains action buttons, status indicators, or other controls relevant to the current view.

Although the footer internally uses ui_hbox_t for layout, it is represented by a dedicated type to:

  • improve readability and prevent accidental misuse
  • allow future extension of footer‑specific behavior
  • provide a clear semantic separation between screen sections

The footer inherits all behavior from ui_hbox_t, including stretching, alignment, spacing, and child widget management.
