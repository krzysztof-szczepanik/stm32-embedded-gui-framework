Detailed Description

Clickable button widget with label and focus styling.

This module defines a simple interactive button widget used in the UI framework. A button consists of:

  • a base widget (ui_widget_t)
  • a label displayed inside the button
  • border and background styling
  • focus‑specific colors
  • an optional on_click callback

The button supports focus handling, drawing, and event processing through its virtual functions.
