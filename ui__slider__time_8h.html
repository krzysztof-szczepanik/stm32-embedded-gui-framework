Detailed Description

Time slider widget (HH:MM:SS formatting).

This widget represents a horizontal slider whose numeric value corresponds to time expressed in minutes. The displayed label automatically formats the value as HH:MM:SS, making it suitable for selecting durations, timers, schedules, or time offsets.

The widget supports:

  • a draggable handle
  • formatted time label
  • configurable range (min, max, step)
  • focus‑dependent styling for border, background and handle

The slider responds to UI_EVENT_LEFT and UI_EVENT_RIGHT to decrease or increase its value.
