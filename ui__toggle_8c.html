Detailed Description

Two‑state toggle widget (left/right switch).

This widget displays two selectable areas (left and right), separated by a vertical divider. Each area contains a label and can be highlighted depending on the selected state.

The widget is interactive: pressing SELECT switches between the left and right side.

It is useful for binary settings such as ON/OFF, YES/NO, ENABLE/DISABLE, etc.
