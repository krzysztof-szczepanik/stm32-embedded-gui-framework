Detailed Description

Key–value label widget (left text + right value).

This widget displays a pair of text labels arranged horizontally: a left label (key) and a right label (value), separated by a vertical separator. It is useful for presenting information in a structured “Label: Value” format (e.g. “Temperature: 21°C”).

The widget is non-interactive and does not handle events.
