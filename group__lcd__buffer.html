void lcd_buffer_init ( void  )

Initialize the LCD framebuffer module.

Sets up background/foreground layers and enables the LTDC line interrupt. Also performs first buffer swap to avoid flickering in foreground layer.

void lcd_swap_buffers ( void  )

Swap the current draw and display buffers (double buffering).

Waits for VSYNC using LTDC line event callback and switches the visible layer with the draw layer. Ensures smooth rendering without tearing.
