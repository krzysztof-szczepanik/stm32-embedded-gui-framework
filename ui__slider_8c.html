Detailed Description

Horizontal slider widget with value display.

This widget represents a horizontal slider with:

  • a draggable handle
  • a numeric value displayed as a centered label
  • configurable range (min, max, step)
  • optional suffix (e.g. "%", "°C")
  • focus‑dependent styling for border, background and handle

The slider responds to UI_EVENT_LEFT and UI_EVENT_RIGHT to decrease or increase its value.
